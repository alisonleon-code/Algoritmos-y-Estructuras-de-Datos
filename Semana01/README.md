# 📘 Prácticas de Arreglos en Java

Repositorio con ejercicios y actividades desarrolladas en Java sobre el uso de arreglos.

---

## 📂 Contenido

### 📁 Balotario01

Ejercicios propuestos:

1. Insertar un elemento en un arreglo ordenado de forma ascendente, manteniendo el orden.
2. Mezclar dos arreglos ordenados en uno nuevo también ordenado.
3. Insertar un valor después del primer elemento que contenga un valor específico.

---

### 📁 Ejercicios (Actividad 1)

1. Vector de 20 notas:

   * Contar aprobados (≥ 12)
   * Contar notas iguales a 17
   * Mostrar el vector
   * Calcular el promedio

2. Suma de vectores:

   * Dos vectores A y B (30 elementos)
   * Resultado en vector C

3. Mostrar vector en orden inverso:

   * Pesos de 10 personas

4. Búsqueda:

   * Vector con valores aleatorios
   * Encontrar el primer número par y su posición
